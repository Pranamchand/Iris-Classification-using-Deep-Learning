# Iris Classification Model

## Overview

This project classifies Iris flowers into three species using machine learning and deep learning techniques. It demonstrates the complete workflow, from data preprocessing and visualization to model training and evaluation.

## Dataset

- **Dataset:** Iris Dataset
- **Target Classes:**
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Label Encoding
- Feature Scaling using StandardScaler
- Perceptron Classifier
- Artificial Neural Network (ANN) using TensorFlow/Keras
- Model evaluation using Accuracy and Classification Report
- Training and validation accuracy visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras

## Results

The project compares the performance of a Perceptron classifier and an Artificial Neural Network for multiclass classification of the Iris dataset. Both models are evaluated using accuracy and classification metrics.
